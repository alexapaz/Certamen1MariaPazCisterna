# portafolio profesional - maría paz cisterna

## descripción del proyecto
este sitio web es un portafolio profesional desarrollado para la asignatura de maquetación web. el proyecto aplica una estética oscura y minimalista para presentar el perfil de una estudiante de analista programador y ciberseguridad.

el objetivo fue construir una página web completa desde cero, utilizando estándares modernos de estructura semántica y diseño adaptable.

## tecnologías utilizadas
* html5: uso de etiquetas semánticas (header, nav, main, section, article, aside, figure, footer).
* css3: diseño basado en flexbox para estructuras lineales y css grid para la galería de proyectos.
* diseño responsivo: implementación de media queries para asegurar la visualización en dispositivos móviles.
* git: control de versiones con un historial de avances ordenado.

## estructura del sitio
1. inicio: banner principal con foto de perfil y títulos destacados.
2. sobre mí: descripción personal con una cita textual y lista de intereses laterales.
3. habilidades técnicas: tabla detallada con los conocimientos y niveles.
4. servicios: tarjetas informativas sobre las áreas de especialización.
5. proyectos: galería de trabajos organizados en una cuadrícula de tres columnas.
6. contacto: formulario con validación para el envío de mensajes.

## bitácora de versiones (commits)
el proyecto se construyó siguiendo estos pasos:
* commit 1: configuración inicial y maquetación de la navegación.
* commit 2: sección de perfil y biografía.
* commit 3: sección de habilidades y servicios.
* commit 4: galería de proyectos.
* commit 5: formulario de contacto y pie de página.
* commit 6: diseño responsivo y adaptabilidad.
* commit 7: metatags y favicon.
